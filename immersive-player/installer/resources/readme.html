System Requirements

  • macOS 10.15 (Catalina) or later
  • GPU with Metal support
  • 4GB RAM minimum (8GB recommended)

HAP Video Format

HAP is a GPU-accelerated video codec that provides excellent playback performance. The following variants are supported:

Variant Use Case
HAP RGB video, smallest files
HAP Alpha RGBA with alpha channel
HAP Q Highest quality, larger files

What Will Be Installed

  • Immersive Player application
  • FFmpeg binaries for HAP video conversion

The application will be installed to your /Applications folder.
